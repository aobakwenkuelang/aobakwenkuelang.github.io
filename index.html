This website serves as documentation for the design system used in my portfolio.

Values

The design of this system was informed by a few key values:


  1. Simplicity: I intend this system to be adaptable. Components should be reusable with little effort.

  2. Approachability: I intend my portfolio to be viewed by as wide a variety of individuals as possible. This means keeping to existing and known design conventions. 'Overly designed' solutions could also cause confusion which would draw away from the primary purpose of my portfolio site.

Foundations

Components

These are the individual elements that make up my portfolio. They include, navigation bars, footers, buttons and cards. Components within the system are meant for use across desktop, tablet and mobile devices.
